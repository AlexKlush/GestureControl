## Что это такое

GestureControl - нативное macOS приложение, которое работает в строке меню и позволяет управлять компьютером жестами рук через встроенную FaceTime-камеру. Никакого дополнительного оборудования не нужно.

Приложение использует **Apple Vision Framework** для распознавания 21 точки на каждой руке в реальном времени, и **CoreGraphics / AppleScript** для управления системой.

## Установка

### Шаг 1 — Скачать

Перейдите в раздел **[Releases](../../releases)** и скачайте последний `GestureControl.dmg`.

### Шаг 2 — Установить

1. Откройте скачанный `.dmg` файл
2. Перетащите `GestureControl.app` в папку **Applications**
3. Закройте окно установщика

### Шаг 3 — Первый запуск

При первом запуске macOS может показать предупреждение *«Разработчик не установлен»*. Это нормально для приложений не из App Store.

**Как обойти:**
- Откройте **Системные настройки → Конфиденциальность и безопасность**
- Прокрутите вниз — найдите блок с GestureControl
- Нажмите **«Открыть всё равно»**

### Шаг 4 — Выдать разрешения

Приложению нужны два разрешения. Без них жесты не работают.

#### 1. Доступ к камере
При первом запуске появится диалог — нажмите **Разрешить**.

Если диалог не появился:
**Системные настройки → Конфиденциальность и безопасность → Камера → включить GestureControl**

#### 2. Универсальный доступ (Accessibility)
Это обязательно для управления курсором и отправки системных событий.

**Системные настройки → Конфиденциальность и безопасность → Универсальный доступ**
→ Нажмите **+** → найдите `GestureControl` в папке Applications → добавьте и включите переключатель.

> После выдачи разрешений перезапустите приложение.

## Жесты

Приложение использует две руки независимо.

### Правая рука

| Жест | Действие |
|---|---|
| ☝️ Указательный палец вытянут | Управление курсором мыши |
| 🤌 Щипок (быстро сжать и отпустить) | Клик мыши |
| 🤌 Щипок (держать > 0.3с) | Перетаскивание (drag) |
| ✊ Сжатый кулак (держать ~0.4с) | Mission Control |
| ➡️ Быстрый взмах вправо | Предыдущий рабочий стол |
| ⬅️ Быстрый взмах влево | Следующий рабочий стол |

### Левая рука

| Жест | Действие |
|---|---|
| 🖕 Только средний палец (тыльной стороной) | Режим «Не беспокоить» |
| ✌️ Указательный + средний (держать ~0.35с) | Пауза / Воспроизведение (F8) |
| ☝️ Только указательный палец | Скролл страницы |
| 🤏 Щипок (расстояние между пальцами) | Регулировка громкости |

**Громкость:** пальцы сведены = тихо, разведены = громко.

## Настройки в строке меню

После запуска в строке меню появится иконка ✋.

| Пункт | Описание |
|---|---|
| Start Tracking | Включить распознавание жестов |
| Stop Tracking | Выключить (камера перестаёт работать) |
| Cursor: ON/OFF | Вкл/выкл управление курсором правой рукой |
| Left Hand: ON/OFF | Вкл/выкл все жесты левой руки |
| Calibrate Gestures... | Открыть мастер калибровки |
| Quit | Закрыть приложение |

> По умолчанию управление курсором **выключено**. Включите через меню если нужно.

## Калибровка

Калибровка адаптирует пороги распознавания под ваши руки и условия освещения. Рекомендуется сделать один раз при первом запуске.

**Меню → Calibrate Gestures...**

Откроется окно с анимированным кольцом прогресса. Приложение последовательно попросит показать 6 жестов по ~2.5 секунды каждый:

1. **Щипок правой** — сведите большой и указательный пальцы
2. **Открытая правая** — обычное положение руки
3. **Кулак правой** — сожмите руку
4. **Средний палец левой** — тыльной стороной к камере
5. **Максимальная громкость** — разведите пальцы максимально
6. **Минимальная громкость** — сведите пальцы вместе

Результаты сохраняются автоматически и применяются при следующем запуске.

## Конфиденциальность

- Видео с камеры **обрабатывается локально** и никуда не передаётся
- Приложение не собирает никаких данных
- Vision Framework работает полностью на устройстве

---

## Технологии

- **AVFoundation** — захват видео с камеры
- **Vision** — распознавание позиций рук (VNDetectHumanHandPoseRequest)
- **CoreGraphics** — генерация событий мыши
- **AudioToolbox** — управление громкостью системы
- **AppleScript / osascript** — переключение пространств, Mission Control
